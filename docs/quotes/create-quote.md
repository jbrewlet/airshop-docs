# Create a Quote

There are three ways to create a quote.

1. **Keyboard shortcuts:** `C` then `Q`
2. **From Inquiries:** Select **Create Quote** in the Quote Grid next to each inquiry
3. **Navigation:** Go to **Quotes** → **Create Quote** in the left navigation
