# Quote Revisions

Revisions are updates you make after a quote is already shared with a customer. A clear revision workflow helps avoid confusion about which version is current.

---

## When to Revise a Quote

Revise when any customer-facing detail changes, such as:

- Scope or line items
- Quantities or pricing
- Labor categories or totals
- Expiration window or project title

---

## Recommended Revision Workflow

1. Open the existing quote and make all required updates.
2. Re-check customer-facing display options in [Quote Options Panel](quote-options-panel.md).
3. Save and review the quote preview before sharing again.
4. Re-send or share the updated quote link with clear context (for example: "Updated with new quantity and pricing").

---

## Communication Tips

- State exactly what changed in the message body.
- Include when the update was made.
- Ask the customer to ignore prior pricing if applicable.

---

## Avoiding Revision Mistakes

- Keep one quote owner when possible.
- Avoid partial edits across multiple sessions; finish updates in one pass before re-sending.
- Verify reply-to destination in [Quote Settings](../quote-settings.md#quote-email-reply-to-settings).
