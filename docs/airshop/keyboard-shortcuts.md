# AirShop Keyboard Shortcuts Guide

AirShop includes keyboard shortcuts for fast navigation and workflow. Press **⌘/** (or **Ctrl+/** on Windows) from anywhere in the app to open the interactive shortcuts panel.

---

## Mac vs Windows

Anywhere you see **⌘** (Command), use **Ctrl** on Windows or Linux.

---

## Main Controls

| Shortcut | Action |
|----------|--------|
| `⌘B` | Toggle Navigation Panel |
| `⌘K` | Focus Search |
| `⌘;` | Open Add New Menu |
| `⌘/` | Show Keyboard Shortcuts |
| `Esc` | Close dialogs / clear search |

---

## Add New Menu

When the Add New menu is open (`⌘;`):

| Shortcut | Action |
|----------|--------|
| `↑` `↓` | Navigate options |
| `Enter` | Select option |
| `Esc` | Close menu |

Available options: Item, Source, Location, Quote, Customer.

---

## Quick Navigation

These shortcuts work from anywhere in the app. Press the keys **one after the other** (no modifier key):

| Shortcut | Action |
|----------|--------|
| `C` then `Q` | Create Quote |
| `G` then `D` | Go to Dashboard |
| `G` then `X` | Go to Inventory Items |
| `G` then `Q` | Go to Quotes |
| `G` then `S` | Go to Settings |
| `G` then `Z` | Go to Inquiries |

---

## Search Modal

When the search modal is open (`⌘K`):

| Shortcut | Action |
|----------|--------|
| `↑` `↓` | Navigate results |
| `Enter` | Open selected result |
| `Esc` | Close search |

---

## Quote Builder

When editing quotes:

| Shortcut | Action |
|----------|--------|
| `⌘E` | Toggle Edit / Preview mode |
| `Tab` | Toggle Edit / Preview (when customer is selected) |

---

## Image Lightbox

When viewing images in the Quote Builder lightbox:

| Shortcut | Action |
|----------|--------|
| `←` `→` | Navigate between images |
| `Esc` | Close lightbox |

---

## Tips

- Shortcuts are disabled when you're typing in a text field to avoid conflicts.
- Sequential shortcuts (like `G` then `D`) have a 1-second timeout—press the keys quickly.
- Works in Chrome, Firefox, Safari, and Edge.
