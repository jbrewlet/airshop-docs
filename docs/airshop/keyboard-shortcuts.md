# AirShop Keyboard Shortcuts

AirShop has helpful keyboard shortcuts. Press **⌘/** (or **Ctrl+/** on Windows) anywhere in the app to see the full legend.

## Mac vs Windows

Replace **⌘** with **Ctrl** on Windows.

## Main Controls

| Shortcut | Action |
|----------|--------|
| ⌘ + B | Toggle Navigation Panel |
| ⌘ + K | Focus Search |
| ⌘ + ; | Open Add New Menu |
| ⌘ + / | Show Keyboard Shortcuts |
| Esc | Close Dialogs / Clear Search |

## Quick Navigation

Type these letter pairs in sequence (no modifier key):

| Shortcut | Action |
|----------|--------|
| CQ | Create Quote |
| GD | Go to Dashboard |
| GX | Go to Inventory |
| GQ | Go to Quotes |
| GS | Go to Settings |
| GZ | Go to Inquiries |

## Search

| Shortcut | Action |
|----------|--------|
| ↑ ↓ | Navigate results |
| Enter | Open selected result |
| Esc | Close search |

## Quote Builder

| Shortcut | Action |
|----------|--------|
| ⌘ + E | Toggle Edit / Preview |
