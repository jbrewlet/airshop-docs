# Quick Start

Get started with AirShop in a few steps.

## Prerequisites

- An AirShop account
- Web browser (Chrome, Firefox, Safari, or Edge)

## Steps

1. Sign in at [airshop.work](https://airshop.work)
2. Complete your profile if prompted
3. Explore the dashboard
